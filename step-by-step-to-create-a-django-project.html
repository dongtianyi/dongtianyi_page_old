step by step to create a django project

二 27 五月 2014
By Tianyi Dong

In Python,django.

tags:

http://www.tylerbutler.com/2012/05/how-to-install-python-pip-and-virtualenv-on-windows-with-powershell/

Start a django project :

1 python

install python

windows: click here to download python.

Linux: sudo apt-get install python

2 download and install virtualenv

if you have pip installed, you can run:

pip install virtualenv

If not, please go to here to download virtualenv and install.

2.1 use pip behind proxy:

pip install --proxy http://cn-proxy.jp.oracle.com:80 -r requirements.txt

do not forget "http://"

3 create virtualenv

virtualenv env-mysite

This will create a dir env-mysite in the current directory. The dir includes all the package and python django project needed.

activate the env:

source env-mysite/bin/activate

windows:

env-mysite/bin/activate

After activate your cmd line change to: (env-mysite).... means that your env activate.

4 start django project:

# install django, you can set version using  pip install Django=1.6.2
pip install Django

# use --template to start a project,
django-admin.py startproject --template=https://github.com/pinax/pinax-project-account/zipball/master mysite

cd mysite

# install database
python manage.py syncdb

# runserver: default run server at: 127.0.0.1:8000
python manage.py runserver

in your browser, input 127.0.0.1:8000, you can see website is running.
